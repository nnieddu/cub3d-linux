# cub3d-linux
Recode a raycasting engine, to create an old style game like Wolfenstein3D.   

Work only on native UNIX OS, using WSL lead to mlx error.

Lib needed :   
sudo apt-get install libx11-dev libxext-dev libbsd-dev
